# Лекция: Управляющие операторы C# — `goto`, `break`, `continue`, `return`

## Введение

В языке программирования **C#** существуют специальные операторы, которые позволяют **управлять потоком выполнения программы**.  
К ним относятся:

- `goto` — безусловный переход;
- `break` — выход из цикла или `switch`;
- `continue` — переход к следующей итерации цикла;
- `return` — выход из метода и возврат значения (если метод не `void`).


## 1. Оператор `goto`

Оператор `goto` выполняет **безусловный переход** к определённой **метке** в коде.

### Синтаксис:

```csharp
goto Метка;

// ...
Метка:
    // выполняемый код
````

### Пример:

```csharp
int i = 0;

start:
Console.WriteLine(i);
i++;

if (i < 5)
    goto start;
```

**Вывод:**

```
0
1
2
3
4
```

> ⚠️ Использование `goto` **не рекомендуется** в большинстве случаев, так как делает код трудночитаемым.
> Лучше использовать циклы и условия.

---

## 2. Оператор `break`

Оператор `break` используется для **прерывания выполнения цикла** (`for`, `while`, `do-while`, `foreach`)
или выхода из блока `switch`.

### Пример 1: Прерывание цикла

```csharp
for (int i = 0; i < 10; i++)
{
    if (i == 5)
        break;

    Console.WriteLine(i);
}
```

**Вывод:**

```
0
1
2
3
4
```

Цикл завершается при достижении `i == 5`.

### Пример 2: Использование в `switch`

```csharp
int day = 3;

switch (day)
{
    case 1:
        Console.WriteLine("Понедельник");
        break;
    case 2:
        Console.WriteLine("Вторник");
        break;
    case 3:
        Console.WriteLine("Среда");
        break;
}
```



## 3. Оператор `continue`

Оператор `continue` **пропускает текущую итерацию** цикла и переходит к следующей.

### Пример:

```csharp
for (int i = 0; i < 5; i++)
{
    if (i == 2)
        continue;

    Console.WriteLine(i);
}
```

**Вывод:**

```
0
1
3
4
```

Когда `i == 2`, тело цикла не выполняется, и программа переходит к следующей итерации.


## 4. Оператор `return`

Оператор `return` используется для **завершения работы метода**.
Если метод возвращает значение, `return` также возвращает это значение вызывающему коду.

### Пример 1: Выход из метода

```csharp
void SayHello()
{
    Console.WriteLine("Привет!");
    return;
}
```

### Пример 2: Возврат значения

```csharp
int Sum(int a, int b)
{
    return a + b;
}

int result = Sum(5, 7);
Console.WriteLine(result);
```

**Вывод:**

```
12
```

### Пример 3: Условный `return`

```csharp
int Divide(int a, int b)
{
    if (b == 0)
    {
        Console.WriteLine("Ошибка: деление на ноль!");
        return 0;
    }

    return a / b;
}
```



## Заключение

Операторы `goto`, `break`, `continue` и `return` позволяют гибко управлять выполнением программы.
Однако при их использовании важно соблюдать **логичность и читаемость кода**.
Особенно это касается `goto`, который может привести к «спагетти-коду».

> 💡 Рекомендация:
> Всегда старайтесь использовать структурированные конструкции (`if`, `for`, `while`) вместо `goto`,
> а `break`, `continue` и `return` применять осознанно для упрощения логики.
