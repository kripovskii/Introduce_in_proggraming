# Многомерные массивы в C#

## 1. Что такое многомерные массивы?

В C# существуют два типа «многомерных» массивов:

| Тип                  | Синтаксис объявления       | Внутренняя реализация         | Название на русском          |
|----------------------|----------------------------|-------------------------------|------------------------------|
| Прямоугольные        | `int[,] arr`               | Один непрерывный блок памяти  | Многомерный массив           |
| Зубчатые             | `int[][] arr`              | Массив указателей на массивы  | Массив массивов (jagged)     |

Самый часто используемый — **прямоугольный** многомерный массив.

## 2. Прямоугольные многомерные массивы

### Объявление и инициализация

```csharp
// 2-мерный массив 3 × 4
int[,] matrix = new int[3, 4];

// 3-мерный массив 2 × 3 × 5
double[,,] cube = new double[2, 3, 5];

// Инициализация с значениями
int[,] arr = {
    { 1, 2, 3 },
    { 4, 5, 6 },
    { 7, 8, 9 }
};
```

### Получение размеров

```csharp
int rows    = matrix.GetLength(0);    // количество строк     (3)
int cols    = matrix.GetLength(1);    // количество столбцов  (4)
int rank    = matrix.Rank;            // количество измерений (2)
int total   = matrix.Length;          // общее количество элементов (12)
```

> `GetLength(dimension)` — правильный способ получить размер по конкретному измерению.  
> `matrix.GetUpperBound(0)` возвращает максимальный индекс (rows-1).

### Доступ к элементам

```csharp
matrix[0, 0] = 10;
int value = matrix[2, 3];
```

### Перебор элементов

```csharp
// Через обычные for
for (int i = 0; i < matrix.GetLength(0); i++)
{
    for (int j = 0; j < matrix.GetLength(1); j++)
    {
        Console.Write(matrix[i, j] + "\t");
    }
    Console.WriteLine();
}

// Через foreach (работает, потому что массив реализует IEnumerable)
foreach (int item in matrix)
{
    Console.Write(item + " ");
}
```

## 3. Зубчатые массивы (jagged arrays)

```csharp
// Массив из 3 массивов разной длины
int[][] jagged = new int[3][];

jagged[0] = new int[] { 1, 2 };
jagged[1] = new int[] { 3, 4, 5, 6 };
jagged[2] = new int[] { 7, 8, 9 };

int value = jagged[1][2]; // 5
```

Зубчатые массивы полезны, когда строки имеют разную длину (например, треугольная матрица).

## 4. Сравнение прямоугольных и зубчатых массивов

| Характеристика            | Прямоугольный `[,,]`         | Зубчатый `[][]`               |
|---------------------------|------------------------------|-------------------------------|
| Память                    | Непрерывный блок             | Массив указателей + отдельные массивы |
| Производительность доступа| Чуть быстрее (одна индексация)| Немного медленнее            |
| Гибкость размеров         | Все измерения фиксированы    | Каждый подмассив может иметь свою длину |
| Удобство объявления       | Проще для фиксированных размеров | Удобнее для «рваных» структур |
| Подходит для математики   | Да (матрицы, тензоры)        | Реже                         |

## 5. Массивы больше двух измерений

```csharp
// 4-мерный массив
int[,,,] hyper = new int[2, 3, 4, 5];

// Пример: цветное изображение с альфа-каналом (высота × ширина × каналы × кадры)
byte[,,,] video = new byte[1080, 1920, 4, 60];
```

Максимальное количество измерений в C# — **32**.

## 6. Полезные методы и трюки

```csharp
// Копирование части массива
int[,] source = { {1,2}, {3,4}, {5,6} };
int[,] dest = new int[2,2];
Buffer.BlockCopy(source, 0, dest, 0, 8 * sizeof(int));

// Преобразование в одномерный массив (по строкам)
var flat = matrix.Cast<int>().ToArray();

// Проверка границ вручную (для скорости в критичных местах)
unsafe
{
    fixed (int* p = matrix)
    {
        int* ptr = p + i * matrix.GetLength(1) + j;
        *ptr = 999;
    }
}
```

## 7. Когда использовать System.Array vs многомерные массивы

| Задача                            | Рекомендация                     |
|-----------------------------------|----------------------------------|
| Матричные вычисления              | `int[,]` или библиотеки (Math.NET, ML.NET) |
| Хранение данных таблицы           | `T[,]`                          |
| Рваные структуры, списки списков  | `T[][]`                          |
| Динамическое изменение размеров   | `List<List<T>>` или `T[,]` + копирование |
| Максимальная производительность   | Прямоугольный массив + `Span<T>`/`Memory<T>` |

## 8. Пример: умножение матриц 3×3

```csharp
int[,] A = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

int[,] B = {
    {9, 8, 7},
    {6, 5, 4},
    {3, 2, 1}
};

int[,] C = Multiply(A, B);

static int[,] Multiply(int[,] a, int[,] b)
{
    int n = a.GetLength(0);
    int[,] result = new int[n, n];

    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            for (int k = 0; k < n; k++)
                result[i, j] += a[i, k] * b[k, j];

    return result;
}
```
