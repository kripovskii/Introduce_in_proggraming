# Условные операторы и циклы в C#

Ниже приведены четыре задания для практики работы с условными операторами и циклами в языке C#. Каждое задание требует ввода числа от пользователя и выполнения логики с использованием `if`, `else if`, `else` и циклов (`for`, `while`).



## Задание 1: FizzBuzz

### Условие:
Запросить у пользователя целое число `n` (гарантируется, что `n > 0`). Используя цикл, вывести числа от `1` до `n` с условиями:

- Если число делится на 3, вывести `"Fizz"`.
- Если число делится на 5, вывести `"Buzz"`.
- Если число делится и на 3, и на 5, вывести `"FizzBuzz"`.
- В остальных случаях — само число.

### Пример вывода:
```bash
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
```

## Задание 2: Пирамида с условиями

### Условие:
Запросить у пользователя целое число `n` (гарантируется, что `n > 0`). Построить "пирамиду" из чисел с `n` строками:

- Каждая строка содержит числа от `1` до номера строки.
- Если номер строки делится на 3 — вместо чисел вывести символ `*`.
- Если делится на 5 — символ `#`.
- Если делится и на 3, и на 5 — символ `+`.

### Пример вывода при `n = 6`:
```bash
1
1 2
* * *
1 2 3 4
# # # # #
```



## Задание 3: Простые числа

### Условие:
Запросить у пользователя целое число `n` (гарантируется, что `n > 1`). Найти и вывести все **простые числа** от `2` до `n`.

**Простое число** — это число, которое делится только на `1` и на само себя.

### Пример вывода при `n = 20`:
```bash
2
3
5
7
11
13
17
19
```
