### Практика работы с методами в C# 

**Тема:** Методы в C# — объявление, перегрузка, params, параметры по умолчанию, обработка краевых случаев.

**Цель:** Закрепить навыки создания перегруженных методов для базовых арифметических операций и вычисления площадей.

**Задача:**

Создайте консольное приложение на C#. В нём реализуйте класс `Calculator` со следующими **public static** методами:

1. **Сложение**  
   Перегруженные методы `Add`:
   - `int Add(int a, int b)` — сумма двух целых чисел.
   - `int Add(int a, int b, int c)` — сумма трёх целых чисел.
   - `int Add(params int[] numbers)` — сумма произвольного количества целых чисел (можно передать 0, 1, 2 и более).  
     Если массив пустой — вернуть 0.

2. **Вычитание**  
   Перегруженные методы `Subtract`:
   - `int Subtract(int a, int b)` — a − b.
   - `int Subtract(int a, int b, int c)` — a − b − c.
   - `int Subtract(params int[] numbers)` — вычитание всех чисел из первого (т.е. numbers[0] − numbers[1] − numbers[2] − …).  
     Если передано меньше 1 числа — вернуть 0. Если ровно одно — вернуть его само.

3. **Умножение**  
   Перегруженные методы `Multiply`:
   - `int Multiply(int a, int b)` — произведение двух чисел.
   - `int Multiply(int a, int b, int c)` — произведение трёх чисел.
   - `int Multiply(params int[] numbers)` — произведение всех переданных чисел.  
     Если передано 0 чисел — вернуть 0. Если одно — вернуть его само.

4. **Деление**  
   Перегруженные методы `double Divide` (возвращаем double, чтобы поддерживать дробный результат):
   - `double Divide(int a, int b)` — a / b. Если b == 0 — вывести сообщение "Деление на ноль!" и вернуть double.NaN.
   - `double Divide(int a, int b, int c)` — a / b / c (с проверкой делителей на ноль).
   - `double Divide(params int[] numbers)` — последовательное деление: numbers[0] / numbers[1] / numbers[2] / …  
     Если передано меньше 2 чисел — вернуть double.NaN и сообщение.  
     При любом делителе равном 0 — прервать вычисление, вывести сообщение и вернуть double.NaN.

5. **Вычисление площади фигуры**  
   Перегруженные методы `CalculateArea` (возвращают double):
   - Круг: `double CalculateArea(double radius)`  
     Если radius < 0 — вывести сообщение и вернуть 0.
   - Прямоугольник: `double CalculateArea(double length, double width)`  
     Если любой параметр < 0 — вернуть 0 с сообщением.
   - Треугольник: `double CalculateArea(double baseLength, double height)`  
     Если любой параметр ≤ 0 — вернуть 0 с сообщением.

**Требования к программе:**

- Все методы — `public static` в классе `Calculator`.
- В методе `Main` продемонстрируйте вызов **каждой перегрузки** каждого метода (включая краевые случаи: пустой params, деление на ноль, отрицательные значения для площади и т.д.).
- Выводите понятные сообщения на консоль, например:
  ```
  Add(5, 7) = 12
  Add(1, 2, 3) = 6
  Add(массив из 4 чисел) = 10
  Add(пустой) = 0

  Subtract(10, 3, 2) = 5
  Multiply(2, 3, 4) = 24
  Divide(20, 4, 2) = 2.5
  Divide(10, 0) → Деление на ноль!

  Площадь круга r=5: 78.53981633974483
  Площадь круга r=-3 → Некорректный радиус!
  Площадь прямоугольника 4x6: 24
  Площадь треугольника (осн=8, выс=5): 20
  ```

- Добавьте комментарии к каждому методу.
- Используйте `Math.PI` для круга.
