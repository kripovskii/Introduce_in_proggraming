
## Практические задания

### 🟢 Уровень 1 — Простое
**Задача:**  
Напишите программу, которая выводит числа от 1 до 10,  
но прерывает выполнение цикла при достижении числа `6`.

**Подсказка:** используйте оператор `break`.

**Ожидаемый результат:**
```

1
2
3
4
5

```

---

### 🟡 Уровень 2 — Среднее
**Задача:**  
Создайте программу, которая находит **первое число**, делящееся на 7 без остатка,  
начиная с переменной `n`.  
Если `n` больше `100`, программа должна завершить выполнение без поиска  
(используйте `return`).

**Пример работы:**
```
Введите число: 10
Результат: 14
```

**Подсказки:**
- Используйте цикл `while`.
- Примените `return` для завершения программы.
- Можно использовать `break` для выхода из цикла после нахождения числа.


---

### 🔴 Уровень 3 — Сложное


**Задача:**  
Напишите программу, которая запрашивает у пользователя числа,  
пока он не введёт `0`.  
Если введено отрицательное число — пропустите его (используйте `continue`).  
Если введено число больше `100` — завершите выполнение программы (`break`).  
После ввода `0` программа должна вывести сумму всех введённых положительных чисел.

**Пример работы:**
```
Введите число: 5
Введите число: -2
Введите число: 12
Введите число: 150
Сумма положительных чисел: 17
```

**Подсказки:**
- Используйте цикл `while(true)`.
- Примените `continue`, чтобы пропустить отрицательные числа.
- Используйте `break`, чтобы завершить программу при `>100`.
- Суммируйте положительные числа в отдельной переменной.
