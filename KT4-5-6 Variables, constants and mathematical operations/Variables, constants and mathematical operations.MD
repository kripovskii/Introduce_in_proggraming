## Что такое переменные?

**Переменная** — это как коробка с названием, где можно хранить данные. У каждой коробки:
- **Имя** (название коробки)
- **Тип** (что можно хранить в коробке)
- **Значение** (что лежит в коробке сейчас)

```csharp
// Пример: создаем коробку с названием "age" для хранения чисел
int age = 25;
```

## Типы данных

### Основные типы:

| Тип | Что хранит | Пример | Размер |
|-----|------------|---------|--------|
| `int` | Целые числа | `10`, `-5`, `0` | 4 байта |
| `double` | Дробные числа | `3.14`, `-2.5` | 8 байт |
| `float` | Дробные числа | `3.14f`, `-2.5f` | 4 байта |
| `decimal` | Точные деньги | `10.99m` | 16 байт |
| `bool` | Логические значения | `true`, `false` | 1 бит |
| `char` | Один символ | `'A'`, `'1'`, `'@'` | 2 байта |
| `string` | Текст | `"Привет"`, `"123"` | зависит от длины |

### Как выбрать тип?
- **Целые числа** → `int`
- **Дроби** → `double` или `decimal` (для денег)
- **Да/Нет** → `bool`
- **Текст** → `string`

## Объявление и инициализация переменных

### 1. Объявление (создаем коробку)
```csharp
int number;      // Создали коробку для чисел
string name;     // Создали коробку для текста
double price;    // Создали коробку для дробей
```

### 2. Инициализация (кладем значение в коробку)
```csharp
number = 10;     // Положили число 10
name = "Иван";   // Положили текст "Иван"
price = 99.99;   // Положили дробное число
```

### 3. Объявление + инициализация (сразу)
```csharp
int age = 25;                   // Число
string city = "Москва";         // Текст
double temperature = -5.5;      // Дробное число
bool isStudent = true;          // true или false
char grade = 'A';               // Один символ
```

## Константы

**Константа** — это переменная, значение которой нельзя изменить после установки.

```csharp
// Обычная переменная (можно менять)
int score = 100;
score = 200; // Так можно

// Константа (нельзя менять)
const double PI = 3.14159;
// PI = 3.15; // ОШИБКА! Нельзя изменить константу

const int MAX_USERS = 100;
const string COMPANY_NAME = "MyApp";
```

### Когда использовать константы?
- Для значений, которые не меняются (π, дни недели, настройки)
- Для магических чисел (чтобы код был понятнее)

## Правила именования

### 1. Правильные имена:
```csharp
string firstName;    // Верно: camelCase
int userAge;         // Верно
double accountBalance; // Верно
const int MAX_SIZE = 100; // Верно: UPPER_CASE для констант
```

### 2. Неправильные имена:
```csharp
int 1number;        // Ошибка: начинается с цифры
string first name;  // Ошибка: пробел
double class;       // Ошибка: ключевое слово
```

### 3. Рекомендации:
- Используйте понятные имена: `userAge` вместо `a`
- Для переменных: `camelCase` (первая буква маленькая)
- Для констант: `UPPER_CASE` (все буквы большие)
- Избегайте сокращений: `firstName` лучше чем `fn`

## Выражения и операции

### Арифметические операции:
```csharp
int a = 10 + 5;    // Сложение: 15
int b = 10 - 5;    // Вычитание: 5
int c = 10 * 5;    // Умножение: 50
int d = 10 / 5;    // Деление: 2
int e = 10 % 3;    // Остаток от деления: 1

// Сокращенные операции:
int count = 10;
count += 5;  // То же что: count = count + 5;
count++;     // Увеличить на 1 (постфиксная)
++count;     // Увеличить на 1 (префиксная)
```

### Операции сравнения:
```csharp
bool result;
result = (5 == 5);  // Равно: true
result = (5 != 3);  // Не равно: true
result = (5 > 3);   // Больше: true
result = (5 < 3);   // Меньше: false
result = (5 >= 5);  // Больше или равно: true
result = (5 <= 4);  // Меньше или равно: false
```

### Логические операции:
```csharp
bool isSunny = true;
bool isWarm = false;

bool andResult = isSunny && isWarm;   // И: false
bool orResult = isSunny || isWarm;    // ИЛИ: true
bool notResult = !isSunny;            // НЕ: false
```

## Преобразование типов

### 1. Неявное преобразование (автоматически)
```csharp
int smallNumber = 10;
double bigNumber = smallNumber; // Автоматически: int → double

// Можно: маленький тип → большой тип
// int → double → decimal
// float → double
```

### 2. Явное преобразование (вручную)
```csharp
double price = 9.99;
int intPrice = (int)price; // Будет 9 (дробная часть отбрасывается)

// Используйте, когда уверены в преобразовании
```

### 3. Преобразование с помощью методов
```csharp
string text = "123";
int number = int.Parse(text);      // Из текста в число: 123
string result = number.ToString(); // Из числа в текст: "123"

// Безопасное преобразование:
string input = "abc";
int value;
bool success = int.TryParse(input, out value); // false - не удалось
```

## Практические примеры

### Пример 1: Калькулятор возраста
```csharp
int birthYear = 2000;
int currentYear = 2024;
int age = currentYear - birthYear;

Console.WriteLine("Ваш возраст: " + age + " лет");
```

### Пример 2: Калькулятор скидки
```csharp
double originalPrice = 1000.0;
double discountPercent = 15.0; // 15%
double discountAmount = originalPrice * (discountPercent / 100);
double finalPrice = originalPrice - discountAmount;

Console.WriteLine($"Цена со скидкой: {finalPrice} руб.");
```

### Пример 3: Проверка пароля
```csharp
string correctPassword = "secret123";
string userInput = "secret123";

bool isPasswordCorrect = (userInput == correctPassword);
Console.WriteLine("Пароль верный: " + isPasswordCorrect);
```

## Частые ошибки

### 1. Использование без инициализации
```csharp
int number;
Console.WriteLine(number); // ОШИБКА! Переменная не имеет значения
```

### 2. Несовместимые типы
```csharp
int a = 10;
string b = "20";
int result = a + b; // ОШИБКА! Нельзя сложить число и текст
```

### 3. Деление на ноль
```csharp
int a = 10;
int b = 0;
int result = a / b; // ОШИБКА! Деление на ноль
```

### 4. Потеря точности
```csharp
double price = 9.99;
int intPrice = (int)price; // Будет 9, а не 9.99
```

## Полезные советы

1. **Всегда инициализируйте переменные** перед использованием
2. **Выбирайте meaningful имена** — код должен читаться как книга
3. **Используйте константы** для magic numbers
4. **Проверяйте типы** перед преобразованием
5. **Комментируйте сложные выражения**

```csharp
// Плохо:
int a = 10;
int b = 5;
int c = a * b + a / b - 3;

// Хорошо:
int baseScore = 10;
int bonusPoints = 5;
int penalty = 3;

// Вычисляем итоговый счет
int totalScore = (baseScore * bonusPoints) + (baseScore / bonusPoints) - penalty;
```

## Заключение

Запомните главное:
- **Переменные** — для хранения изменяемых данных
- **Константы** — для неизменяемых значений
- **Выбирайте правильные типы** под ваши данные
- **Используйте понятные имена** переменных
- **Всегда инициализируйте** переменные перед использованием
