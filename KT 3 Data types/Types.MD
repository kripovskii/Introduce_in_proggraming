# Лекция: Типы данных в C#

## Введение
Типы данных — это основа любого языка программирования. В C# они определяют, какие значения может принимать переменная и какие операции над ними допустимы. Язык C# является статически типизированным, то есть тип переменной известен на этапе компиляции.

---

## Классификация типов данных
В C# все типы данных делятся на две большие группы:

1. **Значимые типы (Value types)**
2. **Ссылочные типы (Reference types)** 

---

## Значимые типы

Значимые типы хранят свои значения непосредственно в памяти.
При присваивании создаётся **копия значения**.

### Основные группы значимых типов:

- **Целые числа**:
  - `byte` (0 до 255)
  - `sbyte` (-128 до 127)
  - `short` (-32 768 до 32 767)
  - `ushort` (0 до 65 535)
  - `int` (-2 147 483 648 до 2 147 483 647)
  - `uint` (0 до 4 294 967 295)
  - `long` (-9 223 372 036 854 775 808 до 9 223 372 036 854 775 807)
  - `ulong` (0 до 18 446 744 073 709 551 615)

- **Числа с плавающей точкой**:
  - `float` (приблизительно 7 знаков после запятой)
  - `double` (приблизительно 15–16 знаков после запятой)
  - `decimal` (высокая точность для финансовых вычислений, до 28–29 знаков)

- **Символьный тип**:
  - `char` (один символ в кодировке Unicode)

- **Логический тип**:
  - `bool` (`true` или `false`)

- **Специальные типы**:
  - `enum` — перечисление (набор именованных констант)
  - `struct` — структура (составной тип данных, похожий на несколько полей)

### Примеры
```csharp
int a = 10;
float b = 3.14f;
bool c = true;
char d = 'A';
decimal price = 99.99m;
````

---

## Ссылочные типы

Ссылочные типы хранят **ссылку на объект в памяти**, а не само значение.
При присваивании копируется ссылка, а не данные.

### Основные ссылочные типы:

* `string` — строка (набор символов Unicode)
* `object` — базовый тип для всех типов данных в C#
* `dynamic` — переменная, тип которой определяется во время выполнения
* Массивы (`int[]`, `string[]` и т.д.)

### Примеры

```csharp
string name = "C#";
object obj = 123;       // boxing
dynamic dyn = "Hello!";
int[] numbers = { 1, 2, 3, 4 };
```

---

## Boxing и Unboxing

C# позволяет преобразовывать значимые типы в ссылочные (boxing) и обратно (unboxing).

```csharp
int x = 42;
object obj = x;     // boxing
int y = (int)obj;   // unboxing
```

---

## var и Типизация

Ключевое слово `var` позволяет компилятору вывести тип переменной автоматически:

```csharp
var num = 100;     // int
var text = "Hi";   // string
```

Важно: тип определяется при компиляции, а не в рантайме.

---

## Nullable-типы

Любой значимый тип может быть объявлен как **nullable** (может хранить `null`):

```csharp
int? value = null;
if (value.HasValue)
{
    Console.WriteLine(value.Value);
}
```

---

## Итог

* Значимые типы хранят данные в стеке и копируются при присваивании.
* Ссылочные типы хранят ссылку на объект в куче.
* В C# предусмотрены как строгие, так и более гибкие механизмы типизации.
* Типы данных помогают компилятору контролировать корректность операций и экономить память.
