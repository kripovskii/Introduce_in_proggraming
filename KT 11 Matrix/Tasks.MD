# Задачи: Одномерные массивы


## Задача 1. Сумма элементов

Дан массив из **N** целых чисел. Найдите сумму всех элементов массива, не используя встроенные функции суммирования (например, `sum()`).

**Пример:**
- Ввод: `[3, 7, -2, 10, 5]`
- Вывод: `23`


## Задача 2. Реверс массива

Напишите программу, которая переворачивает массив **без использования встроенных методов** (например, `reverse()`). Результат должен быть записан в тот же массив.

**Пример:**
- Ввод: `[1, 2, 3, 4, 5]`
- Вывод: `[5, 4, 3, 2, 1]`



## Задача 3. Поиск второго максимума

Дан массив из **N** различных целых чисел. Найдите **второй по величине** элемент массива за один проход (без сортировки).

**Пример:**
- Ввод: `[4, 1, 9, 3, 7]`
- Вывод: `7`




## Задача 4. Циклический сдвиг

Дан массив из **N** элементов и число **K**. Выполните **циклический сдвиг вправо** на K позиций: элементы, «вышедшие» за правую границу, появляются в начале массива.

**Пример:**
- Ввод: массив `[1, 2, 3, 4, 5]`, `K = 2`
- Вывод: `[4, 5, 1, 2, 3]`

> **Подсказка:** учтите случай, когда K больше длины массива.
