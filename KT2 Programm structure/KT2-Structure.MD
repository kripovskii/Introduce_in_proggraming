# Классическая структура программы на C# (.NET Framework 4.7)
Структура программы — это способ организации кода, который определяет:

- Расположение различных элементов кода

- Порядок их следования

- Взаимосвязи между разными частями программы

- Правила оформления и стиль написания кода


## 1. Неизменяемый костяк программы

**Эта структура обязательна и не меняется:**

```csharp
using System;

namespace ИмяВашегоПространстваИмен
{
    class Program
    {
        static void Main(string[] args)
        {
            // ↓↓↓ ВАШ КОД ТОЛЬКО ЗДЕСЬ ↓↓↓


            // ↑↑↑ ВАШ КОД ТОЛЬКО ЗДЕСЬ ↑↑↑
        }
    }
}
```

Представьте, что программа — это книга:

`using` — это оглавление или указатель где что искать

`namespace` — раздел в библиотеке (научная литература, художественная)

`class` — конкретная книга на полке

`method` — глава в книге

`code` — предложения и абзацы внутри главы

Без структуры программа была бы как книга, где все страницы перемешаны, нет глав и абзацев — читать такое невозможно.

## 2. Детальное объяснение каждого элемента

### 2.1 Директивы `using` - подключение библиотек
```csharp
using System; // Самая важная - для ввода/вывода
using System.Collections.Generic; // Для списков и коллекций
using System.Linq; // Для работы с данными
// Другие using вставляются ПЕРЕД namespace
```

### 2.2 Пространство имен `namespace`
```csharp
namespace MyApplication // Можно любое имя
{
    // Все что внутри этих скобок принадлежит пространству имен
    // Помогает организовать код и избежать конфликтов имен
}
```

### 2.3 Класс `Program`
```csharp
class Program // Можно любое имя, но Program - стандарт
{
    // Класс - это контейнер для кода
    // Содержит методы (функции) и поля (данные)
}
```

### 2.4 Метод `Main` - сердце программы
```csharp
static void Main(string[] args)
{
    // static - означает, что метод принадлежит классу, а не объекту
    // void - метод не возвращает значение
    // Main - ОБЯЗАТЕЛЬНОЕ имя (с большой буквы!)
    // string[] args - массив аргументов командной строки
    // {} - внутри этих скобок весь ваш код
}
```

## 3. Что можно менять (ограниченно)

### 3.1 Имя пространства имен
```csharp
namespace CalculatorApp { }    // Для калькулятора
namespace GameApp { }         // Для игры
namespace SchoolProject { }   // для учебного проекта
```

### 3.2 Имя класса
```csharp
class Calculator { }      // Вместо Program
class Game { }           // Вместо Program
class MyApp { }          // Вместо Program
```

## 4. Полный пример с комментариями

```csharp
using System; // Подключаем систему ввода-вывода

namespace MyFirstProgram // Пространство имен
{
    class Program // Главный класс
    {
        // Метод Main - точка входа НЕ МЕНЯЕТСЯ
        static void Main(string[] args)
        {
            // ↓↓↓ ВАШ КОД НАЧИНАЕТСЯ ЗДЕСЬ ↓↓↓

            Console.WriteLine("Добро пожаловать!"); // Вывод текста

            // Объявление переменных
            int number = 10;
            string name = "Иван";

            // Работа с переменными
            Console.WriteLine("Привет, " + name);
            Console.WriteLine("Число: " + number);

            // Ввод данных
            Console.Write("Введите ваше имя: ");
            string inputName = Console.ReadLine();
            Console.WriteLine("Привет, " + inputName);

            // ↑↑↑ ВАШ КОД ЗАКАНЧИВАЕТСЯ ЗДЕСЬ ↑↑↑

            // Ожидание нажатия Enter (чтобы консоль не закрылась)
            Console.ReadLine();
        }
    }
}
```

## 5. Что нельзя менять в классической структуре

### 5.1 Обязательные элементы:
```csharp
// 1. Должен быть класс
class Program // ← обязателен

// 2. Должен быть метод Main
static void Main(string[] args) // ← обязателен

// 3. Должны быть фигурные скобки {}
{
    // 4. Код только внутри метода Main
}
```

### 5.2 Нельзя выносить код за метод Main:
```csharp
class Program
{
    // НЕПРАВИЛЬНО - код вне методов
    Console.WriteLine("Ошибка!"); // ← так нельзя!

    static void Main(string[] args)
    {
        // ПРАВИЛЬНО - код только здесь
        Console.WriteLine("Правильно!"); // ← так можно!
    }
}
```

## 6. Частые ошибки новичков

### 6.1 Отсутствие точки с запятой
```csharp
Console.WriteLine("Привет") // ← ОШИБКА: нет ;
Console.WriteLine("Мир");   // ← ПРАВИЛЬНО
```

### 6.2 Неправильный регистр
```csharp
console.WriteLine("Привет"); // ← ОШИБКА: console с маленькой
Console.WriteLine("Привет"); // ← ПРАВИЛЬНО

main(string[] args);         // ← ОШИБКА: main с маленькой
Main(string[] args);         // ← ПРАВИЛЬНО
```

### 6.3 Код вне метода Main
```csharp
class Program
{
    int number = 10; // ← ОШИБКА: код вне методов

    static void Main(string[] args)
    {
        int number = 10; // ← ПРАВИЛЬНО: код внутри метода
    }
}
```

## 7. Шаблон для копирования

```csharp
using System;

namespace ВашеНазваниеПроекта
{
    class Program
    {
        static void Main(string[] args)
        {
            // Ваш код пишите только здесь


        }
    }
}
```

**Запомните**: Весь ваш код должен находиться только между фигурными скобками метода `Main`!
