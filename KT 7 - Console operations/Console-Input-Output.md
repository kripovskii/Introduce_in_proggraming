# Ввод и вывод информации в консоль на C#

## 1. Основы работы с консолью

В C# взаимодействие с консолью осуществляется с помощью класса `Console`, который входит в стандартную библиотеку `System`.

Основные методы:
- `Console.Write()` — выводит данные без переноса строки.
- `Console.WriteLine()` — выводит данные и переводит курсор на новую строку.
- `Console.ReadLine()` — считывает строку, введённую пользователем.
- `Console.ReadKey()` — ожидает нажатие клавиши и возвращает информацию о ней.

**Пример:**
```csharp
Console.Write("Введите ваше имя: ");
string name = Console.ReadLine();
Console.WriteLine("Привет, " + name + "!");
```

## 2. Перенос строки и управляющие символы

В C# для форматирования строк можно использовать специальные управляющие последовательности (escape-последовательности):

`\n` — перенос строки (новая строка).

`\t` — символ табуляции.

`\\ `— обратный слэш.

`\"` — двойная кавычка.

`\0` — нулевой символ (конец строки в стиле C).

`\a` — звуковой сигнал (beep).


Пример:

```Csharp
Console.WriteLine("Строка 1\nСтрока 2");
Console.WriteLine("Слово\tСлово");
```

Вывод:
```
Строка 1
Строка 2
Слово   Слово
```

## 3. Форматирование строк

### Конкатенация строк

Простейший способ объединения строк — использовать оператор `+`:

```
string name = "Анна";
Console.WriteLine("Привет, " + name + "!");
```

### Интерполяция строк

Интерполяция — современный и удобный способ подставлять значения переменных внутрь строки. Перед строкой ставится знак `$`, а выражения заключаются в фигурные скобки `{}`.

Пример:

```
string name = "Иван";
int age = 20;
Console.WriteLine($"Имя: {name}, Возраст: {age}");
```

Вывод:

```
Имя: Иван, Возраст: 20
```

### Метод string.Format

Альтернативный способ форматирования:

```
string name = "Мария";
int age = 25;
Console.WriteLine("Имя: {0}, Возраст: {1}", name, age);
```

## 4. Пример программы с вводом и выводом

```using System;

class Program
{
    static void Main()
    {
        Console.Write("Введите имя: ");
        string name = Console.ReadLine();

        Console.Write("Введите возраст: ");
        int age = int.Parse(Console.ReadLine());

        Console.WriteLine($"\nРезультат:");
        Console.WriteLine($"Имя: {name}");
        Console.WriteLine($"Возраст: {age}");
    }
}
```
Пример работы:
```
Введите имя: Алексей
Введите возраст: 19

Результат:
Имя: Алексей
Возраст: 19
```
---

5. Итоги

Для вывода информации используются `Console.Write` и `Console.WriteLine`.

Для ввода — `Console.ReadLine` и `Console.ReadKey`.

Управляющие символы `(\n, \t, \\, \")` помогают форматировать текст.

Для форматирования строк лучше всего использовать интерполяцию `($"...")`, так как это наглядно и удобно.
