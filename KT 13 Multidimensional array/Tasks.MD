# Уровень 1 — Основы работы с многомерными массивами в C#

Выполняйте все задания **в методе `Main`**, **без создания отдельных методов**.  
Разрешено использовать только прямоугольные массивы вида `T[,]`.
> ‼️ Загружать эти задания нужно в темы: `Двумерные массивы` и `КТ №7 Разработка программ с использованием многомерных массивов` ‼️

### Задание 1. Матрица 5×5
Создайте двумерный массив `int[5,5]`.  
Заполните его по строкам числами от 1 до 25.  
Выведите массив на экран в виде аккуратной таблицы (элементы выровнены по ширине).

### Задание 2. Транспонирование матрицы
Создайте квадратную матрицу 4×4 и заполните её любыми числами (можно вручную).  
Создайте второй массив того же размера и заполните его так, чтобы он стал транспонированной копией первой матрицы (строки ↔ столбцы).  
Выведите на экран сначала исходную матрицу, затем транспонированную.

### Задание 3. Суммы диагоналей
Создайте квадратную матрицу размером 3×3 или больше и заполните её любыми числами.  
Посчитайте:
- сумму элементов **главной диагонали** (от левого верхнего угла к правому нижнему),
- сумму элементов **побочной диагонали** (от правого верхнего угла к левому нижнему).  

Выведите обе суммы.

### Задание 4. Отражения матрицы
Возьмите любую прямоугольную матрицу (например, 4×4 или 5×6) и заполните её числами.  
Создайте две копии этой матрицы:

1. **Отражение по вертикали** — левый и правый края меняются местами (переворачивание слева направо).  
2. **Отражение по горизонтали** — верх и низ меняются местами (переворачивание сверху вниз).

Выведите на экран:
- оригинальную матрицу,
- матрицу после вертикального отражения,
- матрицу после горизонтального отражения.
