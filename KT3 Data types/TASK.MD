# Задача: Магазин электроники

В магазине продаются товары, и для каждого товара нужно хранить информацию:
- название товара (строка);
- цена за единицу (decimal);
- количество на складе (int);
- признак наличия скидки (bool);
- код категории (символ).

## Условие
Напиши программу на C#, которая:

1. Объявит переменные для товара с типами данных, подходящими по смыслу.
2. Выведет всю информацию о товаре в консоль в читаемом виде.
3. Посчитает общую стоимость всех единиц товара на складе.
4. Если на товаре есть скидка, уменьшит итоговую стоимость на 10%.

## Пример вывода
```
Товар: Смартфон
Цена: 29999.99
Количество: 15
Категория: E
Скидка: Да

Общая стоимость (с учетом скидки): 404999.86
```
---
# Полезная информация
- Для вывода в консоль используйте `Console.WriteLine(Имя переменной)`
- Сохраняйте структуру пронраммы
- Для запуска программы используйте сочетание клавиш `Control + F5`
