# Задача: Конвертер температуры

Пользователь вводит температуру в градусах Цельсия.
Нужно вывести эту температуру в трёх единицах измерения:

- Цельсий (`double`)
- Фаренгейт (`double`)
- Кельвин (`double`)

## Формулы:
- `F = C * 9/5 + 32`
- `K = C + 273.15`

---

## Условие
Напиши программу, которая:

1. Считывает температуру в градусах Цельсия (`double`).
2. Переводит её в Фаренгейты и Кельвины.
3. Выводит результаты на экран.

---

## Пример работы программы
```
Введите температуру в °C: 25
Цельсий: 25
Фаренгейт: 77
Кельвин: 298.15
```

# Полезная информация
Для считывания данных с консоли ииспользуем `Console.ReadLine()` - тип данных `string`
Для других типов данных используем: `Convert` :

- `int` название_переменной = `Convert.ToInt32(Console.ReadLine())`
- `double` название_переменной = `Convert.ToDouble(Console.ReadLine())`
